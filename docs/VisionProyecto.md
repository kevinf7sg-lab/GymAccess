# Visión del Proyecto

## Nombre temporal
GymAccess

## Objetivo

Desarrollar un sistema profesional para gimnasios que permita administrar clientes, asistencias y pagos mediante una tablet de recepción y una aplicación para el administrador.

## Filosofía

El sistema debe ser:

- Simple.
- Rápido.
- Confiable.
- Fácil de usar.
- Escalable.

## Versión 1.0

- Registro de clientes.
- Registro de asistencias mediante QR.
- Control de pagos.
- Credenciales PDF.
- Reportes básicos.
- Configuración del gimnasio.
