# Visión del Proyecto 001

## Nombre
Offline-first

## Objetivo

Desarrollar un sistema profesional para gimnasios que permita funcionar sin internet y ademas tener mas funciones con internet


## Versión 1.0

El cliente podra revisar cuando el dispositivo tablet-celular del gym tenga internet desde su hogar, en cambio cuando el cliente se encuentre el gym y la tablet no tiene internet pero tiene conexion a wifi el cliente podra revisarlo desde su app.
